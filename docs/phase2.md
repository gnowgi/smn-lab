# Phase II — the evo-devo path to nested CAZ modulation

Phase I ends on a result that looks like a failure and is really a signpost.

## The pivot: more transducers do not make more world

Holding the world fixed and adding body segments (hence sensors) in a line **does
not** enrich the world-model — decoding skill is flat across segment count
([S1](experiments/sweep_geometry_worldmodel.md)). And this is not an artifact: it is
exactly what the **resolution principle** predicts — resolution scales with **CAZ
density × internal capacity, not raw transducer count**
([Q1b](experiments/sweep_q1b_resolution.md)). Ungated extra sensors add nothing,
because unmodulated input is dropped.

So the way to a richer world-model is **not** linear scaling of identical zones. It
is *architecture*.

## The path: polarized → tubular → segmented → bilateral → appendicular

Evolution and development do not add sensors in a line; they elaborate the **body
plan**, and each elaboration creates a new level at which zones can modulate other
zones — **nested CAZ modulation**. The progression the SMN monograph (M1) sets out,
and that Phase II makes computational, is:

1. **polarized** — a first axis (head/tail): asymmetry makes a determinate self
   (already visible in the [branched result](experiments/branched_self_model.md):
   symmetric bodies cannot individuate their mirror-parts).
2. **tubular** — linked chains close into a tube: an inside/outside distinction.
3. **segmented** — repeated modules: a within/between-segment hierarchy.
4. **bilateral** — paired zones: left/right differentiation and steering by
   bilateral halt.
5. **appendicular** — branches and limbs: dedicated manipulators, and world-model
   richness that **scales with the network**
   ([E4](experiments/p7_scaling_network.md),
   [E4b](experiments/p8_objecthood_transition.md)).

Each level nests the modulation of the level below — the mechanism is unchanged, but
the *architecture* multiplies what it can construct. That is why the same self-model
function works on chain, tree, and tube (Phase I·①), yet only architectural nesting
— not more identical CAZ — yields the detailed world-models Phase II is about.
